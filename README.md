
# Project Title

A work board is a software, which has tracks your work status. In
general, there are three categories - Todo, In Progress, and Done.


## Features

- Adding and Deleting Sticky Notes
- Searching Notes
- Moving Notes within different categories
- Archiving Notes
- User Friendly UI/UX
